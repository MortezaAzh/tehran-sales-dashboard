# Tehran Sales Dashboard

An interactive GIS dashboard for analyzing cleaning product sales across Tehran neighborhoods.

## Live Demo

Download `tehran_dashboard_standalone.html` and open in any browser — no server required.

## Features

- Interactive map of Tehran with customer locations (Leaflet.js)
- Filter by Product, Brand, Year, and Neighborhood
- Revenue by Product, Brand, Category, and Neighborhood charts
- KPI cards: Total Revenue, Customers, Avg Order Value, Top Neighborhood
- 1,000 synthetic customer records across 25 Tehran neighborhoods

## Products

Degreaser Liquid, Multi-Purpose Liquid, Hand Wash Liquid, Foam Cleaner,
Laundry Liquid, Dish Wash Liquid, Laundry Powder, Bleach, Glass Cleaner

## Tech Stack

- Leaflet.js — interactive map
- Chart.js — charts and visualizations
- Vanilla JavaScript — no frameworks
- Data embedded in HTML — single file, no server needed
